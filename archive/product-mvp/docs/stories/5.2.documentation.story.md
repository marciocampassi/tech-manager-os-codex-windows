### CLI Commands Reference

Full command list (for README.md and user-guide.md):

```bash
tmr --help
tmr --version
tmr init                          # Scaffold vault + generate CLAUDE.md
tmr config                        # Configure AI provider + API key
tmr process                       # AI-powered inbox processing
tmr watch                         # Auto-process inbox on file changes
tmr install <skill-name>          # Install a Claude Code skill
tmr update                        # Update all installed skills
tmr team add <email>
tmr team archive <email>
tmr team list
tmr leadership add <email>
tmr leadership list
tmr project add <name>
tmr project list
```
